# 🤖 AI Daily Briefing Bot

An AI-powered bot that sends a personalized daily email 
every morning with live weather, top news and a smart plan.

## Features
- Live weather updates for your city
- Top news headlines
- AI generated daily plan using Groq AI
- Beautiful HTML email design

## Tech Stack
- Python
- Groq AI API (LLaMA 3.3)
- Gmail SMTP
- wttr.in (weather)

## Setup
1. Clone the repo
2. Run: pip install -r requirements.txt
3. Create .env file with your keys
4. Run: python daily_briefing.py
